Linear Regression model to predict insurance charges
# Insurance Claim Prediction

## Objective
Predict medical insurance charges using Linear Regression.

## Dataset
Medical Cost Personal Dataset

## Approach
- Data preprocessing and encoding
- Linear Regression model training
- Evaluation using MAE and RMSE
- Visualization of BMI, Age, Smoking impact

## Results
- Smoking has the strongest impact on charges
- BMI and age also increase insurance cost
